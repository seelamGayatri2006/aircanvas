Iron Canvas Pro

Hand Gesture Virtual Drawing Application

A real-time virtual drawing project built using Python, OpenCV, MediaPipe, and NumPy.

Features

  • Hand tracking
  • Pinch gesture drawing
  • Color selection palette
  • Neon glow effect
  • Clear canvas option

Technologies Used

Python, OpenCV, MediaPipe, NumPy

Note: The interactive application runs locally because it needs webcam access.
